# 调节功能障碍 · 占位文档

> 占位内容，框架联调用，正式版需替换为眼视光学教科书 / 临床指南。

## 范畴

调节不足、调节不持续、调节痉挛、调节灵敏度下降。长时间近距离用眼（屏幕 / 阅读）人群高发。

## 主诉典型描述

- 近距离工作后视物模糊、视近转视远时聚焦延迟
- 阅读 / 编程数小时后头痛、眉弓压力感
- 部分伴随复视（双眼视轴协同问题）

## 行为学线索

- 注视稳定性 RMS 上升
- 眼动幅度变化与近距离任务节奏不匹配
- 短时间内 saccade 次数显著增多（频繁视线切换补偿）

## 处置原则（占位）

- 调节训练 / 视觉训练
- 屈光全矫，必要时给予近用辅助镜
- 与 CVS / 干眼鉴别诊断
