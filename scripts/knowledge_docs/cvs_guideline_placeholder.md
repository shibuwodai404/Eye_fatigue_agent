# 视频终端综合征（CVS / Computer Vision Syndrome）临床要点 · 占位文档

> 本文件仅为框架联调用的占位内容，待替换为正式临床指南后再发表。

## 定义

视频终端综合征（CVS）指长时间使用电子屏幕引起的一组以视疲劳、视物模糊、眼干涩、头痛等为主的症状群。CVS-Q 问卷为常用筛查工具，总分 6 分以上提示症状显著。

## 主要临床表现

- 眼部：干涩、刺痛、灼烧感、异物感、流泪、畏光、眨眼频次降低、眨眼不完整
- 视觉：视物模糊（远 / 近）、复视、调节疲劳、聚焦困难
- 全身：头痛（额颞部为主）、颈肩僵硬

## 关键行为学指标参考阈值（占位值，正式版需引用文献）

| 指标 | 健康成人参考 | 异常方向 |
|---|---|---|
| 眨眼频率（次/分钟） | 12–20 | < 10 提示长屏使用所致下降 |
| 眨眼完整度比 | > 0.85 | < 0.7 与干眼显著相关 |
| 注视稳定性 RMS（°） | < 0.35 | > 0.5 提示注视维持困难 |
| 眼开合比 | > 0.90 | < 0.85 提示疲劳性闭眼增多 |

## CVS-Q 严重度分级（占位）

- < 6：无显著症状
- 6–10：轻
- 11–15：中
- ≥ 16：重

## 处置原则

1. 行为干预：20-20-20 法则、规律远眺、增加主动眨眼训练
2. 环境调整：屏幕亮度与环境光匹配、屏幕距离 50–70 cm、屏幕略低于水平视线
3. 干眼倾向者：人工泪液、必要时眼科评估
4. 视疲劳显著者：复查屈光，必要时调节功能评估
